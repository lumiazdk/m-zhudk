# 表格导出
[插件下载](https://github.com/lumiazdk/-/tree/master/table2excel "插件")
## 执行

```js
 exportTable() {
      $("#tableExcel").table2excel({
        exclude: ".noExl", //过滤位置的 css 类名
        filename: "CPA" + ".xls", //文件名称
        name: "CPA.xlsx",
        exclude_img: true,
        exclude_links: true,
        exclude_inputs: true
      });
    }
```
